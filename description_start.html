Description de votre projet _ Merci SAM pour le CSS <3

Par Samuel, Robin et Sylvain le 4/03/2020 à 18:46

Enfermés en prison, les détenus ne pensent qu'à leur liberté. Tous rêvent de s'évader, mais combien y parviendront ? Cela dépend de vous...

Dans notre jeu, le joueur incarne un détenu et doit essayer de s'évader de la prison le plus rapidement possible en faisant face à divers obstacles : gardes, chiens policiers, verrous et autres sécurités de la prison.

Trois scénarios de difficulté progressive sont disponibles dans le jeu. Le joueur peut sélectionner le scénario souhaité avant de commencer sa partie.

Nous vous souhaitons bonne chance dans cette aventure, mais surtout agissez dans l'ombre !

Ce travail est réalisé dans le cadre du cours 624-2 Génie logiciel - Partie RIA, donné par MM. Alexandre Cotting et Raphaël Rey. La thématique de ce travail repose sur la création d’un jeu 2D à l’aide de la technologie HTML5.

Le code source de l'application est disponible sur le github du projet.
